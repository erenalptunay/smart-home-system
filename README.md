C++ programlama dili kullanılarak geliştirilmiş kapsamlı bir Akıllı Ev Yönetim Sistemi simülasyonudur.
Proje, Nesne Yönelimli Programlama (OOP) prensiplerine sıkı sıkıya bağlı kalarak; cihaz yönetimi, arıza tespiti, dinamik bildirim sistemi ve güvenlik protokollerini simüle eder.
Genişletilebilirlik ve bakım kolaylığı sağlamak amacıyla endüstri standardı Tasarım Desenleri (Design Patterns) üzerine inşa edilmiştir.

Device Core & Notification Infrastructure -> Base Class Design, Observer Implementation, ID Management
Log & Storage System -> Singleton LogService, File I/O
Controllers & Menu -> Menu Navigation, Device Commands
Security Devices -> Alarm, Detectors
